Reichweitenmessung mit Plausible

Zur Analyse der Seitennutzung setzen wir Plausible Analytics ein, gehostet auf unserem eigenen Server (tracking.server.coral-bytes.com). Aus diesem Grund ist keine Einwilligung (Cookie-Banner) erforderlich.
